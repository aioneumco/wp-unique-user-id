== Description ==
This plugin generates a unique 10-digit ID for each user that registers on your WooCommerce store. The ID is stored in the user meta and is displayed in the user's account dashboard.

== Installation ==
1. Upload the plugin folder `unique-user-id` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Upon registration, a unique 10-digit ID will be automatically assigned to each user.
4. The ID can be viewed by users in their account dashboard.

== Usage ==
- Once the plugin is activated, users will automatically get a unique 10-digit ID when they register.
- The ID is stored in the user meta and can be displayed on the user's account page or used for other purposes as needed.

== Frequently Asked Questions ==

= How can I see the unique ID? =
The unique ID is displayed on the user's account dashboard in WooCommerce.

= Can I customize the length of the unique ID? =
The default ID length is set to 10 digits, but you can modify the code to adjust the length if needed.
